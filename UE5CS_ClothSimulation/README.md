# ComputeShader ClothSimulation  

### Demo效果

编译运行项目后，在编辑器界面内容浏览器选择myAssets/level/CSclothsimulation即可查看布料仿真效果，效果如下所示。

![1](./CS_clsi_demo.gif)

### 实现思路

使用computeshader计算获取布料碰撞后的位置。

使用proceduralmesh创建布料网格。

将位置赋值给mymesh的vertices。
